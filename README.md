# CSC 126 Project Finder

A quiz-based web app that helps CSC 126 students at the College of Staten Island find the perfect raylib graphics project based on their interests and personality.

## Features

- **Personality Quiz** - Answer questions to discover your coding style (based on Holland's RIASEC model)
- **Project Recommendations** - Get personalized C++ raylib project ideas matched to your interests
- **Raylib API Reference** - Browse essential raylib functions with examples
- **Setup Guide** - Step-by-step instructions for setting up raylib on Windows, Mac, and Linux
- **Print & Save** - Export your results as a printable page or text file

## Tech Stack

- React 19
- React Router v7
- Tailwind CSS v3
- Hand-drawn design aesthetic

## Getting Started

```bash
# Install dependencies
npm install

# Start development server
npm start
```

Open [http://localhost:3000](http://localhost:3000) to view the app.

## Build for Production

```bash
npm run build
```

## Project Structure

```
src/
├── components/     # Reusable UI components
├── data/           # Quiz questions and project data
├── pages/          # Page components (Home, Quiz, Results, etc.)
└── index.css       # Global styles and Tailwind config
```
